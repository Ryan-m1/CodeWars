### [🉑35. 搜索插入位置](https://leetcode-cn.com/problems/search-insert-position/)

#### 👉解题思路：

简单基础二分查找题目

**代码：**

```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        int l = 0;
        int r = nums.length - 1;
        while (l <= r) {
            //防止溢出，等同于(l+r)/2
            int mid = l + ((r - l) >> 1);
            if (nums[mid] < target) {
                // target在右区间
                l = mid + 1;
            } else if (nums[mid] > target) {
                // target在左区间
                r = mid - 1;
            } else if (nums[mid] == target) {
                return mid;
            }
        }
        return r + 1;
    }
}
```
